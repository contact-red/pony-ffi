# pony-ffi

A package to provide convenience classes for Pony FFI.

## Status

Experimental, I do not expect existing APIs to change, but new ones to be added.

## Installation

* Install [corral](https://github.com/contact-red/pony-ffi)
* `corral add github.com/contact-red/pony-ffi.git --version 0.0.1`
* `corral fetch` to fetch your dependencies
* `use "pony-ffi/<whichever>"` to include this package
* `corral run -- ponyc` to compile your application
